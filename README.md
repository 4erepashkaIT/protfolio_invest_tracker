# PTX — Трекер инвестиционного портфеля

Легковесное браузерное приложение для отслеживания инвестиционного портфеля: криптовалюты, акции и наличные. Работает полностью на стороне клиента без какого-либо бэкенда.

---

## Возможности

- **Криптовалюты** — поддержка 20 монет (BTC, ETH, SOL, TON и др.) с ценами от CoinGecko
- **Акции** — любой тикер с биржи (AAPL, TSLA, SBER и др.) через Yahoo Finance
- **Наличные** — 8 фиатных валют (USD, EUR, CNY, GBP, AED, TRY, KZT, BYN)
- **Сделки** — добавление покупок и продаж с указанием даты, количества и цены
- **P&L-расчёты** — прибыль/убыток, средняя цена покупки, доходность в % по каждой позиции
- **Аналитика** — диаграммы распределения по активам и типам, доходности
- **Двойная валюта** — все цифры показываются одновременно в USD и RUB
- **Экспорт** — выгрузка истории сделок в CSV
- **Оффлайн-кэш** — последние известные цены сохраняются в localStorage
- **PWA-ready** — поддержка `apple-mobile-web-app-capable` для добавления на главный экран iOS

## Разделы интерфейса

### Портфель (Portfolio)
- Общая стоимость в RUB и USD
- Сводная статистика: стоимость, вложено, прибыль в рублях и USD, ROI %
- Список позиций с текущей ценой, кол-вом, стоимостью и изменением за 24 ч

### Активы (Assets)
- Детальная карточка по каждому активу
- Средняя цена покупки vs текущая цена
- P&L в USD и RUB с иконкой тренда

### Сделки (Transactions)
- Хронологический список всех транзакций
- Фильтр по типу (всё / покупка / продажа)
- Удаление отдельных сделок
- Кнопка экспорта CSV

### Аналитика (Analytics)
- Donut-диаграмма распределения по активам
- Donut-диаграмма по типам (Крипто / Акции / Наличные)
- Горизонтальный bar-chart доходности по позициям

## Хранилище данных

Данные хранятся локально в `localStorage` браузера — ни на какой сервер ничего не отправляется.

| Ключ          | Содержимое                                   |
|---------------|----------------------------------------------|
| `ptx_tx`      | JSON-массив транзакций                        |
| `ptx_prices`  | JSON-объект с кэшем цен `{SYM: {usd, ch24, ts}}` |
| `ptx_usdRub`  | Курс USD/RUB (число)                          |

### Схема транзакции

```javascript
{
  id:       "1713000000000_abc",  // уникальный ID
  op:       "buy" | "sell",
  type:     "crypto" | "stock" | "cash",
  sym:      "BTC",               // тикер
  name:     "Bitcoin",           // отображаемое имя
  geckoId:  "bitcoin",           // ID для CoinGecko (только крипта)
  qty:      0.5,                 // количество
  priceUsd: 60000,               // цена покупки/продажи в USD
  date:     "2024-04-13"         // дата сделки
}
```

## Внешние API

| API               | Endpoint                                                                 | Лимиты          |
|-------------------|--------------------------------------------------------------------------|-----------------|
| CoinGecko (free)  | `GET /api/v3/simple/price?ids={ids}&vs_currencies=usd&include_24hr_change=true` | ~30 req/min     |
| Yahoo Finance     | `GET /v8/finance/chart/{ticker}?interval=1d&range=1d`                   | Без документации|
| ExchangeRate-API  | `GET /v4/latest/USD`                                                     | 1 500 req/month |

Все запросы выполняются параллельно через `Promise.allSettled`. Таймаут каждого запроса — **9 секунд**.

## Алгоритм расчёта позиций

Функция `calcHoldings()` использует метод **средней стоимости** (average cost basis):

1. Транзакции сортируются по дате (от старых к новым)
2. **Покупка**: `qty += amount`, `costUsd += amount × price`
3. **Продажа**: вычитается пропорциональная доля средней стоимости:
   ```
   avgCost = costUsd / qty
   qty     -= sellAmount
   costUsd -= avgCost × sellAmount
   ```
4. Если `qty < 1e-9` — позиция считается закрытой
5. Текущая стоимость, P&L и % доходности рассчитываются на основе актуальных цен

## Ограничения

- **Нет синхронизации** — данные хранятся только в текущем браузере/устройстве
- **Нет аутентификации** — любой, кто имеет доступ к браузеру, видит данные
- **Yahoo Finance API** — неофициальный и может перестать работать без предупреждения
- **CoinGecko free tier** — ограничен по частоте запросов; при превышении вернёт 429
- **Нет истории цен** — приложение показывает только текущую цену и 24 ч-изменение
- **Нет авто-обновления** — цены обновляются только по нажатию кнопки ↻
